# Menú en consola en C#

Programa de consola hecho en C# con 4 opciones:

1. Ingresar nombre y edad y determinar si la persona es menor o mayor de edad  
2. Mostrar tabla de multiplicar del 1 al 13  
3. Mostrar un Hola Mundo  
4. Salir del programa  

## Tecnologías
- C#
- .NET
- Visual Studio
